# Interpolation data

The CSV files are editable templates for concentration-dependent diffusivity and chemical strain. The first column is the dimensionless sodium fraction `xNa`; the second column is SI diffusivity or dimensionless isotropic strain. Values are provisional placeholders, not measurements. Select them with `diffusion.mode=interpolation` or `strain.mode=interpolation`. A selected but missing file is a fatal configuration error.
